✌️Sign-Stack: Bridging Communication Gaps with Computer Vision & Web Intelligence.
Model Link:https://nn698.github.io/Sign-Stack/vision/dashboard/index.html

🌟 THE VISION
   Sign-Stack is an accessibility tool designed to translate sign language into real-time text and speech. 
   By combining a robust python-based web engine with a high fidelity dashboard, this project demonstrates a scalable approach
   to assistive technology in both everyday and industrial environments.

 🚀THE TECHNICAL ARCHITECTURE
  1. The Web Engine: Built with python 3.11,
     Utilising 'Mediapipe' and 'OpenCV' for high speed,
     low latency hand tracking and gesture classification.
  2. The Dashboard: A responsive interface built with HTML5,
     CSS3 and JavaScript, designed to visualize data and
     provide text-to-speech feedback.
  3. The Bridge: Integrated via a local server to ensure real
     time interaction between the hardware(camera) and the
     user interface.

 🔨TECH STACK & SKILLS
  Lanuages: Python, JavaScript, HTML & CSS
  Python Libraries: Mediapipw, OpenCV, CVzone, Numpy.
  Tools: Git/Github(Version Control), VsCode, Virtual Environments(.venv).

  ⚙️HOW TO RUN LOCALLY
    1. Git Clone
    2. Set up the environment
        pip install -r requirements.txt
    3. Launch the engine: Run the main Python Scripts (e.g Python app.py)
      to activate the camera
    4. View the Dashboard: Open index.html on your localhost or with live server 
       to see the live translation
